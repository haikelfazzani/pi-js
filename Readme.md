# PiJs  
Simple playground - Write and run

- [Website repo](https://github.com/haikelfazzani/picode-website)

### Languages
- [x] Javascript (nodejs)
- [x] Typescript (+required: ts-node installed globally)
- [x] Python
- [x] Go

- [x] Format code (javascript/typescript)
- [ ] Autosave
